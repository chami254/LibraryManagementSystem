# Library Management System

## Technologies

- Java 11
- Java Servlets
- JSP
- Maven
- Apache Tomcat 9
- MySQL
- Eclipse IDE

## Features

- User Registration
- User Login
- Book Search
- Borrow Books
- Return Books
- Admin Dashboard
- Book Management
- Borrowing History

## Setup

1. Clone repository
2. Import as Maven Project
3. Update Maven
4. Configure Tomcat 9
5. Create MySQL database
6. Run on server
